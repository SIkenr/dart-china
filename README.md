# Dart-China

Dart中文社区官方App，使用Flutter编写

## 功能

Dart中文社区是使用[Discouse]构建，Discourse本身功能异常丰富，APP目前只支持主题浏览和回复等核心功能

  * 主题列表
  * 主题详情与回复
  * 发表主题
  * 搜索
  * 通知消息
  * 我的
